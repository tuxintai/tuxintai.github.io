头文件

  1. import numpy as np:
    • numpy 是 Python 中用于科学计算的一个重要库。
    • as np 的作用是给 numpy 起一个简短的别名 np，以方便在代码中使用。
    • 这个代码段导入了 numpy 库，让我们可以使用其中的各种功能，比如数组操作、线性代数运算等。
  2. from timeit import timeit:
    • timeit 模块用于执行一小段 Python 代码多次，并测量执行时间。
    • from timeit import timeit 的意思是从 timeit 模块中导入 timeit 函数。
    • 这样做的好处是我们可以直接使用 timeit 函数，而不必使用 timeit.timeit() 的形式。
  3. from decimal import Decimal:
    • Decimal 类用于高精度的十进制计算。
    • 这个代码段导入了 Decimal 类，让我们可以使用高精度计算所需的功能。

  1. xn.shape[0] 是一个 NumPy 数组 xn 的属性，用于获取数组的第一个维度的长度，即数组的行数。

    • 对于一维数组，.shape 返回一个元组，其中包含一个值，表示数组的长度。
    • 对于二维数组，.shape 返回一个元组 (行数, 列数)。

  2. np.arange(N) 是 NumPy 库中的一个函数，用于创建一个从 0 到 𝑁−1 的整数序列。

  3. n.reshape((N, 1)) 是对数组 n 进行重新形状的操作，将其从一维数组变形为二维数组

    • n.reshape((N, 1)) 将 n 变形为一个列向量，其中包含了 N 行和 1 列

  4. np.dot(factor, xn) 是 NumPy 库中的一个函数，用于计算两个数组的点积（内积）。

    • 对于一维数组，np.dot(a, b) 计算的是两个数组的内积。
    • 对于二维数组，np.dot(a, b) 计算的是两个矩阵的乘积。
