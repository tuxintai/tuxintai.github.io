C语言中的unsigned

发表于 2024-05-20 分类于 C语言小知识

unsigned

数据类型修饰符，表示无符号整数，和 int 相比可以增加整数类型最大值

使用原因：

  • 扩大范围

    ​ 无符号需求不需要为负数保留位

  • 逻辑需求

​ 在一些场景如 数组索引 内存地址等，负数没有意义，因此使用无符号整数。
